We can also find the antiderivative of \(f(x) = 2x^{-1} - 6\sqrt{x} + 10 = 2\frac{1}{x} - 6x^{1/2} + 10\) term-by-term. Then the antiderivative is

\begin{align*} F(x) \amp = 2\cdot\ln(|x|) - 6\frac{1}{1/2+1}x^{1/2+1} + 10x + C \\ \amp = 2\cdot\ln(|x|) - 6\frac{1}{3/2}x^{3/2} + 10x + C \\ \amp = 2\cdot\ln(|x|) - 4x^{3/2} + 10x + C \end{align*}

Notice that this is exactly the same answer, and even the exact same computations as in computing \(\intdx{ 2x^{-1} - 6\sqrt{x} + 10 }\text{.}\) The only difference between antiderivatives and indefinite integrals is the notation.
