\begin{align*} \intdx{ 2x^{-1} - 6\sqrt{x} + 10 } = \amp \intdx{ 2x^{-1} } - \intdx{6\sqrt{x}} + \intdx{10} \amp \\ = \amp 2\intdx{x^{-1}} - 6\intdx{\sqrt{x}} + \intdx{10} \end{align*}

For the first integral, because the exponent is \(-1\text{,}\) we cannot use the power rule. Instead, we must rewrite it as a fraction \(x^{-1}=\frac{1}{x}\text{.}\) For the second integral, we do not have a rule for square roots, so we must rewrite it as a power \(\sqrt{x} = x^{1/2}\)

\begin{align*} = \amp 2\intdx{\dfrac{1}{x}} - 6\intdx{x^{1/2}} + \intdx{10} \\ = \amp 2\ln(|x|) - 6\cdot \frac{1}{1/2+1} x^{1/2+1} + 10x + C\\ = \amp 2\ln(|x|) - 6\cdot \frac{1}{3/2}x^{3/2} + 10x + C\\ = \amp 2\ln(|x|) - 4 x^{3/2} + 10x + C \end{align*}
