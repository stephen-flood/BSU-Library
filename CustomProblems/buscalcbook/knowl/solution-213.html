Solution 13.6.b.1.
\begin{align*} f'' \amp = \ddx{f'} = \ddx{x^4 - 4x^3} \\ \amp = 4x^3-12x^2 \end{align*}

\(f''=0\) when

\begin{align*} 4x^3 -12x^2 \amp = 0\\ 4x^2( x - 3)\amp = 0 \end{align*}

\(f''=0\) when \(x=0\) or \(x=3\)

\(f''\) is a polynomial, so it is always defined. Making the sign chart for \(f''\) gives.

The original function \(f\) is concave up on the interal \((3,\infty)\text{.}\) The function \(f\) is concave down on the interval \((-\infty,0)\cup(0,3)\text{.}\)
