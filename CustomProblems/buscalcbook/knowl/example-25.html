Example 10.1. Slope of a Constant Function.

Some functions always output the same number \(c\text{,}\) regardless of the input \(x\text{.}\) These functions have the form \(f(x) = c\text{,}\) and their graphs are horizontal lines.

This function is horizontal everywhere, so has a slope of \(0\) for all \(x\text{.}\) In other words, we have

\begin{equation*} f'(x) = \ddx{c} = 0 \end{equation*}

which is the same as saying

\begin{equation*} [ c ]' = 0 \end{equation*}
