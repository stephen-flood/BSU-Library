In mathematics, functions are often defined by rules, and the domain of a function is usually set of all the inputs where the rule can be applied. For example, if \(f(x) = \frac{1}{x}\text{,}\) then I can apply the rule to any number except zero, because \(\frac{1}{0}\) is undefined (or \(DNE\)). This was discussed in greater detail in Section 1.1 in general, and Exploration 1.4 in particular.
