\begin{equation*} \frac{dh}{dx} = \ddx{ \blue{ \Big( \red{3-\ln(x)}\Big)^{10} } } \end{equation*}

This is a composition of two functions, with the inside function (argument) \(\red{g(x) = 3-\ln(x)} \) and outside function \(\blue{f(u)} = \blue {u^{10}}\)

\begin{align*} \frac{dh}{dx} = \amp \ddx{ \blue{ \Big( \red{3-\ln(x)}\Big)^{10} } } \\ = \amp \ddx{\blue{ ( \red{g(x)} )^{10} } } \\ = \amp \blue{ 10\cdot ( \red{g(x)} )^9 }\cdot \ddx{ \red{g(x)} }\\ = \amp \blue{10\cdot ( \red{3-\ln(x)} )^9 }\cdot \ddx{ \red{3-\ln(x)} }\\ = \amp 10\cdot \Big(3-\ln(x)\Big)^9\cdot \Big( 0 - \frac{1}{x}\Big)\\ = \amp -10 \Big(3-\ln(x)\Big)^9\cdot \frac{1}{x} \end{align*}
