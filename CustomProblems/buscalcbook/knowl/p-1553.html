To find concavity and inflection points, we must work with the second derivative \(f''\text{.}\) First, we must find \(f''\text{.}\)

\begin{equation*} f'(x) = \ddx{x^3 - 9x^2 + 24x - 3 } = 3x^2 -18x +24 \end{equation*}
\begin{equation*} f''(x) = \ddx{3x^2 -18x +24} = 6x - 18 \end{equation*}
