Solution 1.4.a.1.

A fraction \(\dfrac{A}{B}\) is defined as long as the denominator \(B\neq 0\text{.}\)

That means that the fraction \(\dfrac{2}{3-x}\) is defined if and only if the denominator \(3-x \neq 0\text{.}\)

Rewriting this, we see that the the fraction is defined if and only if \(3 \neq x\text{.}\)

To write this in interval notation, draw the points where the function is defined on a number line. Use an open circle at \(x=3\) to exclude that value, and draw a thick line to include all other numbers

We can now read the domain off of our number line. The function is defined on the interval \((-\infty,3)\cup(3,\infty)\)
