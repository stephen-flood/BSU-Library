Solution 1.4.b.1.

A fraction \(\sqrt{\ A\ }\) is defined as long as the denominator \(A\geq 0\text{.}\)

That means that the fraction \(\sqrt{x}\) is defined if and only if the inside \(x \geq 0\text{.}\)

To write this in interval notation, draw the points where the function is defined on a number line. Use a closed circle at \(x=0\) to include that value, and draw a thick line to the right to include all positive numbers.

We can now read the domain off of our number line. The function is defined on the interval \([0,\infty)\)
