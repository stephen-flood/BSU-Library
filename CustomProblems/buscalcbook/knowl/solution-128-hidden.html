We are given the starting quantity \(x_1 = 5\) and the change in quantity \(\Delta x = 5\text{.}\) Using this, we can reason backward to get the ending quantity \(x_2 = 105\text{.}\) We can use the AROC equation

\begin{equation*} \dfrac{\Delta C}{\Delta x} = \dfrac{C(105) - C(100)}{105-100} = \dfrac{7352.5 - 7000}{5} = 70.5 \end{equation*}

or create an AROC table

\(x_1\) \(x_2\) \(\Delta x\) \(y_1\) \(y_2\) \(\Delta y\) \(\Delta y/\Delta x\)
\(100\) \(105\) \(\color{red} 105 - 100 = 5\) \(\color{blue} C(100) = 7000\) \(\color{blue} C(105) = 7352.5\) \(\color{blue} C(105) - C(100) \color{green} = 7352.5 - 7000 = 352.5\) \(\dfrac{\color{green} 362.5}{\color{red} 5} = 70.50 \)
