There is a consistent strategy when using the generalized power rule in all of the examples above.

  1. See a composition \(h(x) = \blue{ f\Big(\red{g(x)}\Big) } \text{.}\)

  2. Identify the argument \(\red{g(x)}\) and the outside function \(\blue{f(u)} = \blue{u^m}\text{.}\)

  3. Find the derivative of the outside

    \begin{equation*} \blue{f'(u)} = \blue{m\cdot u^{m-1}} \end{equation*}

  4. Apply the rule

    \begin{equation*} \ddx{\blue{ ( \red{g(x)} )^{m} } } = \blue{ m\cdot ( \red{g(x)} )^{m-1} }\cdot \ddx{ \red{g(x)} } \end{equation*}

An easy way to summarize this is to write

\begin{equation*} \ddx{ \blue{ f\Big(\red{g(x)}\Big) } } = \blue{f'\Big( \red{g(x)}\Big)}\cdot \red{g'(x)} \end{equation*}

But is this more general formula true for any outside function \(g(x)\text{?}\)
