Solution 4.7.d.1.

Recall that \(P\cdot (1+ r)^t\) gives the result of increasing the value \(P\) by a rate of\(r\text{,}\) repeated \(t\) times.

We have shown above that the future value is given by

\begin{equation*} F(t) = 1000\cdot \left( 1.03041595691 \right)^{t} \end{equation*}

Rewriting this we get

\begin{equation*} F(t) = 1000\cdot \left( 1 + 0.03041595691 \right)^{t} \end{equation*}

In other words, this is equivalent to increasing the value by an actual, or effective, rate of \(3.041\dots\) percent each year.
