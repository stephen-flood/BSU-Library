Complete List of Derivative Rules.

Suppose that \(f,g\) are differentiable functions of \(x\text{,}\) that \(m,c\) are constant numbers. Then

  • \(\displaystyle \D{c}=0\)

  • \(\D{m\cdot x+b}=m\) and \(\D{x} = 1\)

  • \(\displaystyle \D{x^m}=m\cdot x^{m-1}\)

  • \(\displaystyle \D{ c\cdot f(x)}=c\cdot \D{f(x)}\)

  • \(\displaystyle \D{f(x) \pm g(x)} = \D{f(x)} \pm \D{g(x)}\)

  • \(\displaystyle \D{ e^x } = e^x \)

  • \(\displaystyle \D{ \ln(x) } = \frac{1}{x}\)

  • \(\displaystyle \Big[f\cdot g\Big]' = f\cdot \Big[g\Big]' + g\cdot \Big[f\Big]' = f\cdot g' + g\cdot f' \)

  • \(\displaystyle \left[\dfrac{t}{b}\right]' = \dfrac{b\cdot t' - t\cdot b'}{(b)^2} \)

  • \(\D{ f(g) } = f'(g)\cdot g'\)

    • \(\displaystyle \D{ \big(g(x)\big)^m } = m \big(g(x)\big)^{m-1}\cdot g'\)

    • \(\displaystyle \D{ e^{g(x)} } = e^{g(x)}\cdot g'\)

    • \(\displaystyle \D{ \ln\big(g(x)\big) } = \dfrac{1}{g(x)}\cdot g'\)

  • If \(b\) is any constant number, \(\D{ b^x } = b^x\cdot \ln(b)\)
