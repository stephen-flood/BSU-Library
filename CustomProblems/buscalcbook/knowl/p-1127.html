The only rule that can handle the coefficient of \(3\) is the last one: \(\dfrac{d}{dx}\Big[ {\color{red}c}\cdot {\color{blue}f(x)} \Big] = {\color{red} c}\cdot \dfrac{d}{dx}\Big[\color{blue} f(x)\Big]\) That means we must interpret the problem as follows, first using the coefficient rule, and then using the power rule`

\begin{align*} f'(x) = \amp \dfrac{d}{dx}\Big[{\color{red} 3}\cdot {\color{blue}x^2} \Big]\\ = \amp {\color{red} 3} \cdot \ddx{{\color{blue}x^2} } \\ = \amp 3 \cdot 2\cdot x^{2-1}\\ = \amp 6\cdot x^{1}\\ = \amp 6\cdot x \end{align*}
