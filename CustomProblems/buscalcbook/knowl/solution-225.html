Solution 14.3.c.1.

Again, we can either find the antiderivative of \(f(x) = \frac{5}{x} = 5\frac{1}{x}\) as before using our guess/check/correct approach, or we can use our indefinite integral formulas.

\begin{align*} \intdx{5\frac{1}{x} } = \amp 5\intdx{\frac{1}{x}}\\ = \amp 5\cdot \ln(|x|) + C \end{align*}
