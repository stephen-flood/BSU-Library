To get more comfortable with what this means, the graph of the piecewise function \(f(x) = \begin{cases} 1 + x & \text{ if } x \lt 2 \\ 1 & \text{ if } x = 2 \\ -1+0.5x & \text{ if } x \gt 2 \\ \end{cases}\) is given below. As you change the value of \(x\text{,}\) the left side of the screen displays

  1. The current value of \(x\text{.}\)

  2. Whether \(x\) is greater, equal, or less than \(2\text{.}\) This is used to decide which of the three equations to use when computing \(f(x)\text{.}\)

  3. The result of plugging the current value of \(x\) into the equation for \(f(x)\text{,}\) which gives the current \(y\) value.
