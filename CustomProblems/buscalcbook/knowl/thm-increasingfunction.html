Theorem 12.4.

Suppose that \(f\) is a differentiable function at \(x\)

  1. height \(f\) is increasing (\(\,\nearrow\,\)) at \(x\) if and only if the slope \(f'(x)\) is positive \((+)\)

  2. height \(f\) is decreasing (\(\,\searrow\,\)) at \(x\) if and only if the slope \(f'(x)\) is positive \((-)\)
