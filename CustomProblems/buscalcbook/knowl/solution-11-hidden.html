At first, this problem can be confusing, because it doesn't have any fractions or roots. But actually, that makes this example very easy.

We are able to multiply, add, and subtract any real number. In other words, the function is defined for all inputs. To write this in interval notation, fill in the entire real line.

We can now read the domain off of our number line. The function is defined on the interval \((-\infty,\infty)\)
