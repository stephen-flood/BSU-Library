Solution 12.4.a.1.

In this example, we are given a factored version of the derivative \(f'\text{,}\) so we do not need to use any of our rules.

Next, we must find the points where \(f'=0\) or \(f'\ DNE\text{.}\) Recall that a fraction \(\dfrac{A}{B}=0\) if and only if \(A=0\text{,}\) and \(\dfrac{A}{B}\ DNE\) if and only if \(B=0\text{.}\) We must therefore solve two equations.

For the first,

\begin{align*} f' = \amp 0\\ (x+3)e^x \amp =0 \end{align*}

Because \(e^x \neq 0\) for all \(x\text{,}\) this can only happen if \(x+3=0\text{,}\) which means when \(x=-3\text{.}\)

For the second,

\begin{align*} f' \amp DNE \\ (x-2^2) \amp =0 \\ x-2 \amp =0\\ x \amp = 2 \end{align*}

Recall that we were given the derivative function, which we now use to check the sign of each interal. We can either use a calculator as before

\begin{align*} f'(-4) \amp = \dfrac{(-4+3)e^{-4}}{(-4-2)^2} = \dots = (-)\\ f'(0) \amp = \dfrac{(0+3)e^0}{(0-2)^2} = \dots = (+)\\ f'(3) \amp = \dfrac{(3+3)e^3}{(3-2)^2} = \dots = (+) \end{align*}

or we can do basic "sign arithmetic" in our head

\begin{align*} f'(-4) \amp = \dfrac{(-)(+))}{(-)^2} = \frac{(-)}{(+)} = (-)\\ f'(0) \amp = \dfrac{(+)(+)}{(-)^2} = \frac{(+)}{(+)} = (+)\\ f'(3) \amp = \dfrac{(+)(+))}{(+)^2} = (+) \end{align*}

Either way, we obtain

Reading off of the sign chart for \(f'\text{,}\) we see that the original (mystery) function \(f\) is increasing on the intervals \((-3,2)\cup(2,\infty)\text{,}\) and the original (mystery) function \(f\) is decreasing on the interval \((-\infty,-3)\text{.}\)
