We know that a product \(a\cdot b= 0\) if and only if \(a=0\) or \(b=0\text{.}\) So to solve the equation

\begin{equation*} -0.4x^2 + 10x = 0 \end{equation*}

since we have 0 on the right, we should try factoring the left side. Because both terms have an \(x\text{,}\) we can factor \(x\) out of both terms.

\begin{equation*} x\cdot (-0.4x + 10 ) = 0 \end{equation*}
