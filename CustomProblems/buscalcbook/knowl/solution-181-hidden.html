This is a composition with argument \(\red{g(x)} = \red{1-2x^3}\) and outside function \(\blue{f(u)} = \blue{ \ln(u) }\text{.}\)

Using our regular derivative rules, we know that \(\red{g'(x)} = \red{ \ddx{1-2x^3} } = \red{-6x^2} \) and that \(\blue{f'(u)} = \blue{\ddu{ \ln(u) }} = \blue{\dfrac{1}{u}} \)

\begin{align*} F'(x) = \amp \ddx{\blue{ \ln\big(\red{1-2x^3}\big) } } \\ = \amp \ddx{ \blue{f\Big( \red{g(x)} \Big) } }\\ = \amp \blue{f'\Big( \red{g(x)}\Big)}\cdot \red{g'(x)} \end{align*}
