Suppose that \(f(x)\) and \(g(x)\) are differentiable functions. Then

\begin{equation*} \ddx{ f(x) + g(x) } = \ddx{f(x)} + \ddx{g(x)} \end{equation*}

and

\begin{equation*} \ddx{f(x) - g(x) } = \ddx{f(x) } - \ddx{g(x)} \end{equation*}
