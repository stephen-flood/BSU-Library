First, we must find the slope function using the power rule

\begin{equation*} f'(x) = \ddx{x^4} = 4 \cdot x^{4-1} = 4 x^3 \end{equation*}

In bracket notation, this is

\begin{equation*} f'(x) = [x^4]' = 4 x^{4-1} = 4x^3 \end{equation*}

Next, we can use the slope function to find the slope at \(x=10\).

\begin{equation*} \text{slope at 10} = f'(10) = 4 \cdot 10^3 = 4000 \end{equation*}
