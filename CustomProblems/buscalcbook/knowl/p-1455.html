Suppose you are given a height function \(f\text{.}\) To find the absolute maximum height and minimum height of \(f\) on the interval \([l,r]\text{,}\)

  1. Find the derivative (slope) function \(f'(x)\)

  2. Find all critical points \(c\) where \(f'(c) = 0\) (horizontal tangents) or \(f'(c)\ DNE\) (singluar points).

  3. Find the height of the function at each of the critical points \(c\)

  4. Find the height of the function at the two endpoints (find \(f(l)\) and \(f(r)\)).
