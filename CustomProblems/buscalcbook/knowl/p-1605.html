We want to find a function whose derivative is \(x^5\text{.}\)

Guess 1

Guess: The derivative of any power function is a power function. So our first guess might as well be something like

\begin{equation*} F=x^5\text{.} \end{equation*}

Check:

\begin{equation*} F'(x) = \ddx{x^5} = 5x^4 \red{\ \neq\ x^5} \end{equation*}

Taking the derivative reduces the power by one, so if we want to end up with \(x^5\text{,}\) we must start out with a \(x^6\) term.

Guess 2:

Guess: Now let's try guessing \(F(x)=x^6\)

Check:

\begin{equation*} F'(x) = \ddx{x^6} = 6x^5 \red{\ \neq\ x^5} \end{equation*}

We now have the correct power, but the wrong coefficient. To fix our guess, we should start out with a coefficient that will cancel out the coefficient of \(6\text{.}\)

Guess 3:

Guess: \(F(x) = \frac{1}{6}x^6\)

Check:

\begin{align*} F'(x) \amp =\ddx{ \frac{1}{6}x^6 } \\ \amp =\frac{1}{6} \ddx{x^6}\\ \amp = \frac{1}{6} 6 x^5\\ \amp = x^5 \end{align*}

That is the function we wanted after the derivative. Therefore \(F(x) = \frac{1}{6}x^6\) is the ante derivative.
