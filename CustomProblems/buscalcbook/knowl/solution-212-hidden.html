\begin{align*} f' \amp = \ddx{ \frac{1}{5}x^5 - x^4}\\ \amp = \frac{1}{5}5x^4 - 4x^3\\ \amp = x^4 - 4x^3 \end{align*}

\(f'=0\) if and only if

\begin{align*} x^4-4x^3 \amp =0\\ x^3(x-4) \amp =0 \end{align*}

\(f'=0\) if \(x=0\) or \(x=4\text{.}\)

\(f'\) is a polynomial so there are no values where \(f'\ DNE\text{.}\)

\(f\) has a local maximum at \(x=0\text{,}\) and a local mininmum at \(x=4\text{.}\)
