Theorem 12.2.

A function \(f\) can only have a maxima or minima at \(x=a\) if one of the following are true

  1. \(f\) is horizontal at \(a\text{.}\) This happens when \(f'=0\text{.}\)

  2. \(f\) has a singularity at \(a\text{.}\) This happens when \(f'\ DNE\text{.}\)

  3. \(a\) is an endpoint of the height function \(f\text{.}\)
