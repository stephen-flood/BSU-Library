In the expression

\begin{equation*} F(t) = 1000\cdot \left( 1 + \frac{0.03}{12}\right)^{12 t} \end{equation*}

the base \(1 + \frac{0.03}{12} = 1.0025 \) is a single number. That lets us rewrite the expression as

\begin{equation*} F(t) = 1000\cdot \left( 1.0025 \right)^{12 t} \end{equation*}

Next, we can use the property of exponents \(a^{r\cdot s} = {\left(a^{r}\right)}^s\) to rewrite the expression as

\begin{equation*} F(t) = 1000\cdot \left( 1.0025^{12} \right)^{t} \end{equation*}

Now we can again simplify the base \(1.0025^{12} = 1.03041595691\text{,}\) which gives the expression

\begin{equation*} F(t) = 1000\cdot \left( 1.03041595691 \right)^{t} \end{equation*}

We have now written \(F(t) = a \cdot b^t\) with \(a=1000\) and \(b = 1.03041595691\text{.}\)
