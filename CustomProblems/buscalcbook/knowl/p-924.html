There were a lot of parts of this calculation. You can either use a the definition \(\dfrac{\Delta C}{\Delta x} = \dfrac{C(x_2) - C(x_1)}{x_2 - x_1} \) which is a single formula that combines all of these steps, or you can organize the different parts separately using the table below. This is called an Average Rate of Change table, often abbreviated AROC.

\(x_1\) \(x_2\) \(\Delta x\) \(y_1\) \(y_2\) \(\Delta y\) \(\Delta y/\Delta x\)
\(100\) \(150\) \(50\) \(7000\) \(10750\) \(3750\) \(75\)
