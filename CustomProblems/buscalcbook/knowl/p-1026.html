To use the limit definition,

  1. Completely simplify the fraction using the specific function \(f\) and number \(a\) that you are given.

  2. Compute the limit of the simplified function by letting \(h\) go to \(0\text{.}\) For the purposes of this course, this means plugging in \(h=0\) as long as this does not result in division by \(0\text{.}\)
