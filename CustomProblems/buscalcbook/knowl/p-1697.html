Using indefinite integral notation, we can proceed more directly

\begin{align*} \intdx{\frac{1}{\sqrt[3]{x}}} = \amp \intdx{ x^{-1/3}} \end{align*}

Because \(\frac{-1}{3}\neq -1\text{,}\) we can use the indefinite integral formula \(\intdx{x^m} = \frac{1}{m+1}x^{m+1} + C\)

\begin{align*} \phantom{\intdx{ \frac{1}{ \sqrt[3]{x} } }} = \amp \dfrac{1}{-1/3+1} x^{-1/3+1} \\ = \amp \dfrac{1}{2/3} x^{2/3} + C \\ = \amp \dfrac{1}{0.66\dots} x^{0.66\dots} \end{align*}
