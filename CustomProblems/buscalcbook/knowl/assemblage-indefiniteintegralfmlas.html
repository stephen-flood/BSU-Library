Indefinite Integral Formula List.

  • If \(m\neq -1\text{,}\) \(\intdx{ x^m } = \frac{1}{m+1}x^{m+1} + C \)

  • \(\displaystyle \intdx{ \frac{1}{x} } = \ln(|x|) + C \)

  • \(\displaystyle \intdx{ e^x } = e^x + C \)

  • If \(a\) is a constant, \(\intdx{a} = ax + C\)

  • If \(a\) is a constant, then \(\intdx{a\cdot f(x)} = a \cdot \intdx{f(x)}\)

  • \(\displaystyle \intdx{f(x) + g(x)} = \intdx{f(x)} + \intdx{g(x)}\)
