We want to find the percent \(r\) so that

\begin{equation*} 234 = 300\cdot (1 + r) \end{equation*}

Solving for \(r\) gives

\begin{equation*} 1 + r = \frac{234}{300} = 0.78 \end{equation*}
\begin{equation*} 1 + r = 0.78 \end{equation*}
\begin{equation*} r = -0.22 \end{equation*}

The value has decreased by 22%
