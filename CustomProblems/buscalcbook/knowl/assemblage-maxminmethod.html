A Strategy for Finding Extrema.

Let \(f\) be any function which is continuous on an open interval. To find the local extrema,

  1. Use the derivative function \(f'\) to find where \(f'=0\) and when \(f'\ DNE\text{.}\)

    • Polynomials: factor to find the 0's.

    • Fractions: \(\dfrac{A}{B}\) is equal to zero if when \(A=0\) and is undefined when \(B=0\text{.}\)

  2. Plot these points on a number line. This cuts the real numbers into several intervals.

    Use a closed dot for \(f'=0\) and an open circle for \(f'\ DNE\text{.}\)

  3. Compute \(f'\) on each of the intervals you have found above. Mark whether \(f'\) is \((+)\) or \((-)\) below the interval, and draw \(\nearrow\) or \(\searrow\) above the interval.

  4. Read off the answer to the question, using your sign chart as a very rough sketch of the function.
