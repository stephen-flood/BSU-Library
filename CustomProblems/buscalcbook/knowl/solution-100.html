Solution 5.1.b.1.

We are given that \(\color{blue} f({\color{red} x}) = 2{\color{red} x} + 1\text{.}\) We have also found that \(f^{-1}(x) = 0.5x - 0.5\text{.}\)

To compute \({ \color{blue} f\Big({\color{red} f^{-1}(x)}\Big) }\text{,}\) replace every \(\color{red} x\) in the equation for \(\color{blue} f({\color{red} x}) \) with the equation \(\color{red} f^{-1}(x) = 0.5x - 0.5\text{.}\)

\begin{align*} { \color{blue} f\Big({\color{red} f^{-1}(x)}\Big) }= \amp {\color{blue} f\Big( { \color{red} 0.5x - 0.5}\Big)}\\ = \amp {\color{blue} 2\cdot \Big({ \color{red} 0.5x - 0.5}\Big) + 1}\\ = \amp 2\cdot 0.5x - 2\cdot 0.5 + 1\\ = \amp 2\cdot 0.5x - 2\cdot 0.5 + 1\\ = \amp x - 1 + 1\\ = \amp x \end{align*}

We have shown in this case that \(f\Big(f^{-1}(x)\Big)=x\text{.}\) In other words, we have shown that the function and inverse undo eachother.
